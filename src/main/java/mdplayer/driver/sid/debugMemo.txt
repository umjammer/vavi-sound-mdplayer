SIDのビルド手順メモ
　msys64 を使用し以下の様にビルドする。
　
　libsidplayfpをビルド
autoconf
./configure
make install

　sidplayfpをビルド
autoconf
./configure
make

一旦ビルドできれば、
後は解析用のコードを入れつつビルドして確認を繰り返す。
　
cd ../..
cd libsidplayfp-2.0.4/
make install
cd ..
cd sidplayfp-2.0.1/
make
cd src
./sidplayfp.exe --resid -ri -f44100 Artillery.sid


------------------------


チップ自体のエミュレーションは
  libsidplayfp\builders\resid-builder\reSID\
内の
  sid.cs
あたり。

   write()
でチップにデータ書き込み。
   clock()
などclockで始まるメソッドでレンダリング。
